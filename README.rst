This is RAC Demo project

This project uses- 

1. angular 1.5.7
2. webpack module bundler
3. jade
4. sass
5. bootstrap
6. karma - mocha

To Set up this project -

1. Clone this repository on your desktop 
2. Run npm install
3. Run npm install webpack-dev-server

Build & Run Instructions -

1. npm run build
2. npm run dev
